What is it?
{% br 2 %}
autosyntax turns simple words into python code.
{% br 2 %} {# L I S T M I L K E G G S N U T E L L A#}
For example:
list milk eggs nutella
becomes
[ 'milk' , 'eggs' , 'nutella' ]
{% br 3 %}
Similarly,
dict meaning 42 happy pizza
{ 'meaning' : 42 , 'happy' : 'pizza' }
{% br 7 %}
{# C L A S S E S#}
It can also do more complicated stuff,
like classes:

class Dog Animal init fluffy .True args
turns into >
class Dog(Animal):
   def __init__ ( self , fluffy= True, *args):
     self .fluffy = fluffy
     super (). __init__ ()

How much time would it take you to type the end result?
{% br 7 %}
It needs only 1 global hotkey to do everything,
but it's also flexible.
