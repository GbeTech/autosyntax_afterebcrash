How to use

General behaviour

autosyntax knows what to do based on "Operators".

Operators are almost always built-in words from the python standard library, to keep a 0 learning curve (e.g list, def, class and so on).

In a given line, the Operator is the first word.


With the basic Operators, autosyntax converts words to strings, unless marked with a dot.

In which case, the word remains as-is to be used as a variable.
list milk nutella
[ 'milk' ,  'nutella' ]
list milk .nutella
[ 'milk' ,  nutella]
