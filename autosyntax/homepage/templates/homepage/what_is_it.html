What is it?

autosyntax turns simple words into python code.


For example:
list milk eggs nutella
becomes >
[ 'milk' , 'eggs' , 'nutella' ]



similarily,
dict meaning 42 happy pizza
becomes >
{ 'meaning' : 42 , : 'happiness' : 'pizza' }







It can also do more complicated stuff, like:

class Dog Animal init fluffy True
turns into >
class  Dog(Animal):
   def  __init__  ( self ,  fluffy= True ):
     self .fluffy = fluffy
     super (). __init__ ()

check out the inheritance, the default fluffy value, the automatic self, super() etc.







it needs only 1 global hotkey to do everything,
but it's also flexible.
